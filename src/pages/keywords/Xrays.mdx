# Röntgenstrahlung

#### Röntgenstrahlung ist, wie sichtbares Licht auch, eine elektromagnetische Welle. Sie ist für das menschliche Auge unsichtbar. Im Gegensatz zu sichtbarem Licht besitzt die Röntgenstrahlung eine kleinere <a href="/keywords/wavelength-energy-frequency/">Wellenlänge</a> und ist energiereicher. Dadurch kann sie Materie durchdringen, weswegen sie z.B. in der <a href="/keywords/medical-use-xray/">Medizin</a> bei verschiedenen bildgebenden Analyseverfahren eingesetzt wird. 

Auch in der Physik wird Röntgenstrahlung verwendet, etwa um die chemischen Eigenschaften eines Materials oder die Struktur von Kristallen zu bestimmen. Ein weiterer wichtiger Anwendungsbereich ist die Bestimmung der Elementzusammensetzung, also wie viel von Element A und wie viel von Element B an einer bestimmten Position in der Probe vorhanden sind. Das kann mit Hilfe der <a href="/keywords/xrf">Röntgenfluoreszenzanalyse (RFA)</a> getan werden kann. Diese Methode wird - auch wegen ihrer Zerstörungsfreiheit - in der <a href="/keywords/archeometry/">Archäometrie</a> angewendet.

Benannt wurde die Röntgenstrahlung nach ihrem Entdecker ‘Wilhelm Conrad Röntgen’, der diese mysteriöse Strahlung das erste Mal 1895 beobachtete, als <a href="/keywords/fluorescence/">Fluoreszenz</a> Gegenstände nahe einer experimentellen Kathodenstrahlröhre zu leuchten begannen. Das Besondere hierbei, die Röhre hatte er vorher mit schwarzer Pappe abgedeckt, so dass sichtbares Licht nicht entweichen konnte. Er erkannte als Erster die Bedeutung dieser neuartigen Strahlung und beschrieb sie wissenschaftlich. Eine der ersten Röntgenaufnahmen zeigt die Hand seiner Frau, bei der nur die  <a href="/keywords/medical-use-xray/">Knochen des Handskeletts</a> zu sehen sind. Das weichere Gewebe um die Knochen herum (Haut, Muskeln, Sehnen) wird von der Röntgenstrahlung leichter durchdrungen.

Mittlerweile weiß man, dass diese Strahlung organische Materie verändern kann und damit z.B. Krankheiten wie Krebs hervorrufen kann. Deswegen gibt es spezielle Regelungen, die die gesetzlich zulässigen Grenzwerte (Strahlungsdosen) festlegen.
