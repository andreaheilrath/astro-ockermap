# Röntgenfluoreszenzanalyse (RFA, XRF)

#### Die Röntgenfluoreszenz beschreibt die <a href="/keywords/Absorption-emission">Emission</a> von <a href="/keywords/Xrays/">Röntgenlicht</a> eines Materials nach vorangegangener Anregung des Materials. Diese Anregung findet in Form von Absorption von Strahlung in den Materialien statt. Dabei besitzt jedes Atom andere, charakteristische Linien, die angeregt werden können. 

Das physikalische Prinzip der RFA basiert auf der Anregung von Elektronen in ein höheres Energieniveau. An dieser Stelle wo das Elektron jetzt fehlt entsteht ein "Loch".
In dieses kann nun ein anderes Elektron, unter aussenden von Licht fallen. Dieses Licht besitzt eine ganz bestimmten <a href="/keywords/Wavelength-energy-frequency/">Wellenlänge</a>, die spezifisch für diesen Übergang (also den Wechsel von einem Elektron aus einer Schale in eine Andere) in diesem Element ist. Somit besitzt jedes Atom ein eindeutig zu diesem Atom zuzuordnendes <a href="/keywords/Fingerprints/">Spektrum</a> (<a href="/keywords/Fingerprints/">Fingerabdruck</a>) aus verschiedenen Übergängen (Linien). 

Die Bestimmung dieser Fingerabdrücke kann mit dem speziellen Messverfahren, der <a href="/posts/XRF/">Röntgenfluoreszenzspektroskopie</a>, gemacht werden. Hierbei werden Röntgenstrahlen von einer <a href="/keywords/Xray-tube/">Röntgenröhre</a> auf eine Probe fokussiert. Diese dringen in die Probe ein und regen dort verschiedene Elektronenübergänge in den einzelnen Atomen an. Nach einer gewissen Zeit, die Lebensdauer genannt wird, fallen die angeregten Elektronen der Atome wieder zurück unter dem Aussenden von Licht mit genau der Energie dieses Übergangs. Ein Röntgendetektor misst dann genau, wie viel Licht (Intensität) welcher Energie von der Probe ausgesendet wird.

Durch Verschieben der Probe kann man so einen großen Bereich abrastern und bekommt an jedem Punkt die Informationen über die Zusammensetzung des Materials.
Schaut man sich nun die Intensität in Abhängigkeit von der jeweiligen Energie an, erhält man das sogenannte Spektrum dieser Probe. Die Spektren unserer Ockerpigmente findest du <a href="/keywords/Measurements-and-evaluation">hier</a>.

Mit dieser Methode kann man die Elementzusammensetzung einer Probe relativ leicht messen und quantifizieren. Allerdings misst diese Methode immer ALLE Schichten einer Probe auf einmal. Um also herauszufinden, ob ein Gemälde z.B. übermalt wurde, wird eine andere Technik benötigt, die uns ermöglicht, Informationen aus einer bestimmten Tiefe der Probe zu erhalten. Im Fall der Untersuchung des <a href="/posts/The-Reading-Hermit">Gemälde “The Reading Hermit”</a> wurde dafür die <a href="/keywords/3D-µ-XRF">konfokale 3D Micro-RFA</a> verwendet.
