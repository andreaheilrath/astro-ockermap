# Röntgenfluoreszenzanalyse (RFA)

#### Die Röntgenfluoreszenz beschreibt die <a href="/keywords/Absorption-emission">Emission</a> von <a href="/keywords/Xrays/">Röntgenlicht</a> eines Materials nach vorangegangener Anregung des Materials. Diese Anregung findet in Form von Absorption von Strahlung in den Materialien statt.
Das physikalische Prinzip der RFA basiert auf der <a href="/keywords/Fluorescence/">Fluoreszenz</a>. Durch Anregung mit Röntgenlicht wird ein kernnahes Elektron aus dem Atom entfernt. An dieser Stelle wo das Elektron jetzt fehlt entsteht ein "Loch".
In dieses kann nun ein anderes Elektron, unter aussenden von Licht "fallen". Dieses Licht besitzt eine ganz bestimmte <a href="/keywords/Wavelength-energy-frequency/">Wellenlänge</a>, die spezifisch für diesen Übergang (also den Wechsel von einem Elektron aus einer Schale in eine Andere) in diesem Element ist. Somit besitzt jedes Atom ein eindeutig zu diesem Atom zuzuordnendes <a href="/keywords/Fingerprints/">Spektrum</a> (<a href="/keywords/Fingerprints/">Fingerabdruck</a>), in dem diese Übergänge zu sehen sind. 

Die Bestimmung dieser Fingerabdrücke kann mit dem speziellen Messverfahren, der <a href="/posts/XRF/">Röntgenfluoreszenzspektroskopie</a>, gemacht werden. Hierbei werden Röntgenstrahlen von einer <a href="/keywords/Xray-tube/">Röntgenröhre</a> auf eine Probe fokussiert. Dieser Fokus liegt im Mikrometerbereich, was dünner ist als der Durchmesser eines Haares. Diese fokussierte Strahlung dringen in die Probe ein und kreiert dort Löcher (fehlende Elektronen) in den einzelnen Atomen. Nach einer gewissen Zeit fallen dann Elektronen einer höheren Schale der Atome unter dem Aussenden von Licht genau dieser Differenzenergie in diese Löcher. Ein Röntgendetektor misst dann genau, wie viel Licht (Intensität) welcher Energie von der Probe ausgesendet wird.

Durch Verschieben der Probe kann man so einen großen Bereich abrastern und bekommt an jedem Punkt die Informationen über die Zusammensetzung des Materials.
Schaut man sich nun die Intensität in Abhängigkeit von der jeweiligen Energie an, erhält man das sogenannte Spektrum dieser Probe. Die Spektren unserer Ockerpigmente findest du <a href="/keywords/Measurements-and-evaluation">hier</a>.

Mit dieser Methode kann man die Elementzusammensetzung einer Probe relativ leicht messen und quantifizieren. Allerdings misst diese Methode immer ALLE Schichten einer Probe auf einmal. Um also herauszufinden, ob ein Gemälde z.B. übermalt wurde, wird eine andere Technik benötigt, die uns ermöglicht, Informationen aus einer bestimmten Tiefe der Probe zu erhalten. Im Fall der Untersuchung des <a href="/posts/The-Reading-Hermit">Gemälde “The Reading Hermit”</a> wurde dafür die <a href="/keywords/3D-µ-XRF">konfokale 3D Mikro-RFA</a> verwendet.
