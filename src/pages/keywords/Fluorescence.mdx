# Fluoreszenz

<Figure url="/src/imgs/keywords/fluoreszenz.png" text='test caption' />
&nbsp;

#### Fluoreszenz ist die <a href="/keywords/Absorption-emission">Emission </a> von Licht eines Materials nach vorangegangener <a href="/keywords/Absorption-emission">Absorption</a> von Licht durch dieses Material. Da das ausgesendete Licht dabei eine Übergangs- und elementspezifische Energie besitzt (<a href="/keywords/Fingerprints/">Fingerabdrücke</a>), kann die Fluoreszenz als Methode in der Materialforschung oder auch z.B. in der <a href="/keywords/Archeometry/">Archäometrie</a> eingesetzt werden. Wird das ganze dann im Bereich der <a href="/keywords/Xrays/">Röntgenstrahlung</a> gemacht, spricht man von der <a href="/posts/XRF/">Röntgenfluoreszenzanalyse (RFA)</a>.
