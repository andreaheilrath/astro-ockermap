# Fluoreszenz

Fluoreszenz beschreibt die (spontane) <a href="/keywords/Absorption-emission">Emission </a> von Licht eines Materials nach vorangegangener <a href="/keywords/Absorption-emission">Absorption</a> von Licht. Da das ausgesendete Licht dabei eine Übergangs- und elementspezifische Energie besitzt (<a href="/keywords/Fingerprints/">Fingerabdrücke</a>), kann die Fluoreszenz als Methode in der Materialforschung oder auch z.B. in der <a href="/keywords/Archeometry/">Archäometrie</a> eingesetzt werden. Wird das Ganze dann im Bereich der <a href="/keywords/Xrays/">Röntgenstrahlung</a> gemacht, spricht man von der <a href="/posts/XRF/">Röntgenfluoreszenzanalyse (RFA)</a>.

*Im **Röntgenbereich** besitzt das einfallende Licht so viel <a href="/keywords/Wavelength-energy-frequency/">Energie</a>, dass das Elektron nicht im Atom bleibt, sondern vom Atom getrennt wird. Dort wird das Atom dann ionisiert und es entsteht ein Loch in einer kernnahen Schale. Wenn in dieses Loch nun ein Elektron aus einer höheren Schale fällt, spricht man hier von Fluoreszenz.*
